# devPTIpack: Priority Targeting Index analysis toolkit

Computes, visualises, and explores Priority Targeting Indices (PTI) over
administrative geographies. The package ships a Shiny application
launched via \[launch_pti()\] together with the Shiny-free pipeline
orchestrator \[run_pti_pipeline()\].
