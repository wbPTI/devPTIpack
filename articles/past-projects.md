# Past Projects

The Project Targeting Index (PTI) framework has been applied in a number
of World Bank analytical and operational projects across more than 30
countries. Below is a catalogue organised by region. The **App** column
links to the live dashboard; **App repo** and **Data repo** link to the
corresponding GitHub repositories on the
[wbPTI](https://github.com/wbPTI) organisation.

**Legend**

| Symbol | Meaning                                     |
|--------|---------------------------------------------|
| ✅     | App confirmed publicly accessible           |
| 🔒     | Requires World Bank internal network or VPN |
| ❌     | App URL returns 404 — no longer hosted      |
